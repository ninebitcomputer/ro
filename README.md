# Language

## Expressions

### Binary

`*`: Multiplication
`/`: Division

`+`: Addition
`-`: Subtraction

### Unary

`-`: Negation
`+`: Plus

## Statements

```
Type ident [ = Expression ] ;
if Expression Block [ else if Block ] [ else Block ] ;
```

### Blocks

```
{ [ Statement* ] }
```
